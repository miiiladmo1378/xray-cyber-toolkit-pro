# ⚡ Xray Cyber Toolkit Pro

## 🚀 Overview
Xray Cyber Toolkit Pro is a lightweight, single-file web tool for managing Xray / V2Ray configurations and executing server operations via VPN/Proxy.

Runs fully in the browser with no installation required.

---

## 🎯 Core Purpose
- Manage Xray / V2Ray configs  
- Generate ready-to-use commands  
- Simplify server workflows  
- Execute operations via VPN  

---

## 🔥 Features
- Supports vless / vmess / trojan / ss  
- Extract full config details  
- Generate config.json and commands  
- Install apt / pip / run bash scripts  
- Execute commands via VPN  
- Connection monitoring & logs  

---

## ⚡ Optimization
